Actualizar Lista de Productos

Suba un archivo CSV con la información de productos.

Importante: El archivo CSV debe contener al menos las siguientes columnas:
  • # (número secuencial)
  • CODIGO (código de referencia)
  • DESCRIPCION (descripción del producto)
  • Precio Antes de Iva
  • Dscto (descuento)

Otras columnas útiles pero opcionales: Marca, Origen, Ref, Vehiculo, Bod, AGOTADO/ESTADO.

Archivo de productos actual: No hay archivo

Actualizar Lista de Clientes

Suba un archivo CSV con la información de clientes.

Importante: El archivo CSV debe contener al menos las siguientes columnas:
  • NIT CLIENTE (número de identificación tributaria)
  • NOMBRE (nombre o razón social)
  • ESTABLECIMIENTO (nombre del establecimiento)
  • DIRECCION (dirección física)
  • TELEFONO (número de contacto)
  • DESCTO (porcentaje de descuento)
  • CLI_CIUDAD (ciudad)
  • CLI_EMAIL (correo electrónico)
  • ID ASESOR (identificación del asesor asignado)
Archivo de clientes actual: No hay archivo

Actualizar Lista de Asesores

Suba un archivo CSV con la información de asesores.

Importante: El archivo CSV debe contener al menos las siguientes columnas:
  • ID (número de identificación o cédula)
  • NOMBRE (nombre completo del asesor)
  • ZONA (zona o territorio asignado)
  • MAIL (correo electrónico)
  • CEL (número de teléfono celular)
Archivo de asesores actual: No hay archivo

Actualizar Lista de Correos por Área

Suba un archivo CSV con la información de los correos que recibirán copia de los pedidos.

Importante: El archivo CSV debe contener las siguientes columnas:
  • AREA (departamento o sección)
  • MAIL (correo electrónico)

Nota: Los pedidos se enviarán siempre al cliente y al asesor que realizó el pedido, además de los correos configurados aquí.
